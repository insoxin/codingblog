== Description ==

中文汉字验证码为两个随机生成的独立汉字。

* 可以设置在哪些页面使用验证码，在评论页可以设置根据用户等级关闭验证码。
* 可以设置是否开启评论页验证码的**ajax验证**功能。
* 1.0版小疏忽，请重新下载，已更新，欢迎大家测试。

* Chinese captcha is two randomized chinese characters.Setting to hide the captcha from logged in users or admins.
* Setting to show the captcha on the forms for comments, registration, lost password, login, or all.
* You can enable ajax function in comment form or not.

环境要求：

* 最好是wordpress3.0以上版本，低版本未做测试
* 推荐PHP5.0以上版本
* 开启GD2扩展
* 开启session支持

Requirements/Restrictions: 

* Wordpress 3.0 or higher
* PHP5 is highly recommended
* Enable GD2 extension
* Enable session

== Installation ==

1. Search 'Chinese Captcha' through the 'Add new plugins' page in Wordpress
1. 在 Wordpress 的'添加新插件' 页里搜索 'Chinese Captcha'
2. Select 'install plugin', and select 'active plugin' after downloaded
2. 选择 '安装插件'，然后再选择 '启用插件'
3. Effect immediately or customized settings
3. 可以直接使用，也可以自定义设置
Or 
或者
1. Download and uncompress 'chinese-captcha' package to the '/wp-content/plugins/' directory
1. 下载并解压 'chinese-captcha' 压缩包到 '/wp-content/plugins/' 目录下
2. Activate the plugin through the 'Plugins' menu in Wordpress
2. 在 Wordpress 的 '插件' 菜单项里启用
3. Effect immediately or customized settings
3. 可以直接使用，也可以自定义设置
